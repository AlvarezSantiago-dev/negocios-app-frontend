# Client (React) — Servidor Negocios App

Scaffold mínimo creado con Vite + React + Axios.

Cómo arrancar:

```powershell
cd client
npm install
npm run dev
```

Notas:

- La API por defecto apunta a `http://localhost:8080/api`. Cambia `REACT_APP_API_URL` si tu backend está en otro puerto.
- Asegúrate de arrancar el backend antes de usar el cliente.
